# Idea Validation

> 사이드 프로젝트 아이디어 검증 파이프라인

> **공유 안내:** 이 아이디어 모음은 팀 브레인스토밍 공유용입니다. 특정 아이디어의 사업화·구현·지식재산권(IP) 귀속은 별도 합의로 정합니다. 검증 점수/판정은 작성 시점의 1차 분석일 뿐, 확정된 의사결정이 아닙니다.

## 이 폴더에서 다루는 것

`/idea` 스킬로 분석된 사이드 프로젝트 아이디어의 검증 결과를 기록합니다.
각 아이디어는 5축 분석(구현 가능성, 수익화, 시장 수요, 차별화, 지속 가능성)을 거쳐 PASS/CONDITIONAL/HOLD/REJECT 판정을 받습니다.

## 판정 기준

| 판정 | 조건 | 의미 |
|------|------|------|
| PASS | 평균 7+ AND 최저 5+ | 바로 구현 착수 |
| CONDITIONAL | 평균 5+ AND 최저 3+ | 조건 해결 시 진행 |
| HOLD | 평균 4-5 | 아이디어 보완 필요 |
| REJECT | 평균 <4 OR 최저 2 이하 | 현 상태 진행 비추천 |

## 흐름

```
/idea "아이디어" → 분석 문서 생성 → PASS → /spec → /execute → 프로젝트 등록
```
