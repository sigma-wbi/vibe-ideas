# ChatGPT (OpenAI)

> 가장 넓은 생태계를 가진 AI 만능 도구

## 한줄 요약

가장 큰 사용자 기반과 서드파티 생태계를 보유. 텍스트, 이미지, 음성, 코드를 아우르는 범용성이 최대 강점이다.

## 강점 (이럴 때 쓰면 좋다)

- **범용성** — 글쓰기, 코딩, 분석, 대화 등 모든 영역에서 고르게 강함
- **이미지 생성** — DALL-E 통합으로 대화 중 이미지 생성/편집 가능
- **GPT Store & 플러그인** — 수만 개의 커스텀 GPT와 서드파티 도구 연동
- **추론 모델 (o-시리즈)** — o3, o4-mini 등 깊은 사고가 필요한 수학/과학/코딩 문제에 특화
- **Computer Use** — GPT-5.4 기준 OSWorld 75%. 화면을 보고 컴퓨터를 직접 조작
- **음성 대화** — Advanced Voice Mode로 자연스러운 실시간 음성 대화

## 약점 (이럴 때는 피하자)

- **비용** — Pro 플랜 $200/월. API도 타사 대비 비싼 편
- **깊은 코딩 에이전트** — Claude Code 수준의 자율 코딩 에이전트는 아직 성숙도가 낮음
- **일관성** — 모델이 다양해서 어떤 모델이 언제 쓰이는지 파악하기 어려울 수 있음
- **지시 따르기** — 복잡한 형식 지정 시 Claude 대비 이탈률이 높은 편

## 실전 팁

### 시스템 프롬프트 전략
- Custom Instructions와 Memory 기능을 활용하면 매번 설정하지 않아도 일관된 응답
- o-시리즈 사용 시 "reasoning effort"를 조절할 수 있음 (low/medium/high)

### 이 모델에서만 잘 되는 것
- **GPT Store** — 특정 업무용 커스텀 GPT를 검색해서 바로 사용
- **Canvas** — 문서/코드를 별도 편집 창에서 실시간 협업 편집
- **Deep Research** — Pro 전용. 자율적으로 웹을 탐색하며 심층 리서치 보고서 작성
- **DALL-E 이미지** — 대화 맥락을 이해한 이미지 생성/수정

### 다른 모델로 전환해야 할 신호
- 대규모 코드베이스를 한 번에 분석해야 할 때 → Claude (1M 컨텍스트 + 코드 특화)
- Google 서비스 연동이 필요할 때 → Gemini
- API 비용을 최소화해야 할 때 → Grok (가격 대비 성능 최고)
