# Grok (xAI)

> 실시간 세계를 아는 AI — 가격 대비 성능의 다크호스

## 한줄 요약

X(Twitter) 실시간 데이터에 접근하고, 업계 최저 수준의 API 가격을 제공하는 가성비 최강 모델. 2M 토큰 컨텍스트로 대용량 처리도 가능.

## 강점 (이럴 때 쓰면 좋다)

- **실시간 정보** — X(Twitter) 피드에 직접 접근. "지금 트렌딩 토픽", "이 사람이 최근에 뭐라고 했어?" 등 실시간 질의 가능
- **압도적 가격** — Grok 4.1 Fast 기준 $0.20/$0.50 per 1M 토큰. GPT-4o 대비 10배 이상 저렴
- **초대형 컨텍스트** — Grok 4.1 Fast가 2M 토큰 컨텍스트 지원 (업계 최대)
- **환각 감소** — Grok 4.1 기준 환각률 65% 감소 (12% → 4.2%). 신뢰성 크게 향상
- **필터링 최소화** — 다른 모델이 거부하는 주제에도 비교적 유연하게 응답

## 약점 (이럴 때는 피하자)

- **생태계 미성숙** — 플러그인, 커스텀 봇 등 서드파티 생태계가 가장 작음
- **에이전트/도구 연동** — MCP, function calling 등 에이전트 기능이 타사 대비 미성숙
- **X 의존성** — 실시간 정보 강점이 X 플랫폼에 편중. 일반 웹 검색은 다른 모델과 비슷
- **코딩 특화 도구 부족** — Claude Code, GitHub Copilot 같은 전문 코딩 에이전트 없음
- **구독 구조 복잡** — X Premium, Premium+, SuperGrok 등 플랜이 혼재

| 모델 | 포지션 | 컨텍스트 | API 가격 (입력/출력, /1M 토큰) |
|------|--------|---------|------|
| **Grok 4** | 플래그십 — 복잡한 추론 | 131K | $3 / $15 |
| **Grok 4.1 Fast** | 고속 균형 — 초대형 컨텍스트 | 2M | $0.20 / $0.50 |

## 실전 팁

### 시스템 프롬프트 전략
- "Fun mode"와 "Regular mode" 전환 가능. 업무용은 Regular 추천
- 실시간 정보 요청 시 "현재", "오늘", "최근" 등 시간 키워드를 명시하면 X 데이터를 더 적극 활용

### 이 모델에서만 잘 되는 것
- **X 실시간 분석** — 특정 토픽/인물의 최신 트윗 트렌드 분석
- **대용량 저비용 처리** — 2M 컨텍스트 + 최저가 조합으로 대량 데이터 배치 처리에 최적
- **언필터드 응답** — 다른 모델이 거부하는 창작/풍자/논쟁적 주제에 대한 응답

### 다른 모델로 전환해야 할 신호
- 정밀한 코드 에이전트가 필요할 때 → Claude
- 이미지 생성이 필요할 때 → ChatGPT 또는 Gemini
- Google 서비스 연동이 필요할 때 → Gemini
- 안정적인 서드파티 생태계가 필요할 때 → ChatGPT
