# AI Design & Build Tools

> 자연어 → 디자인 → 실행되는 코드. Figma 시대 다음의 도구들.

## 이 카테고리에서 다루는 것

LLM이 직접 텍스트 답변을 주는 단계를 넘어, **자연어 → 작동하는 UI/앱**으로 가는 도구들. 2026년 들어 Figma 중심의 워크플로우가 빠르게 재편되고 있어 별도 카테고리로 분리했다.

- **대화형 디자인**: Claude Design, Claude Artifacts (자연어 → 디자인/프로토타입)
- **AI 앱 빌더**: v0, Bolt, Lovable (자연어 → 실행되는 React/Next 앱)
- **AI 친화 디자인 시스템**: shadcn/ui (AI가 가장 잘 뱉는 컴포넌트 라이브러리)

## 트렌드 한 줄 (2026년 5월 기준)

> **Graduate Workflow**: 탐색(Artifacts/v0) → 검증(Bolt/Lovable) → production(Cursor/Claude Code).
> 한 도구로 끝까지 가는 시대가 아니라, **단계마다 도구를 바꾸는 게 정상**.
