## 상황별 추천 — "이 작업에는 이 모델"

### 개발자 업무

| 상황 | 추천 | 이유 |
|------|------|------|
| 코드베이스 전체 리팩토링 | **Claude** | 1M 컨텍스트 + 최상위 코딩 성능 + 에이전트 |
| 빠른 프로토타입 / 스크립트 | **ChatGPT** | 범용 + 빠른 반복 + 플러그인 |
| 코드 리뷰 & 버그 분석 | **Claude** | 구조적 분석 + 정확한 지시 따르기 |
| API 문서 읽고 구현 | **Gemini** | 무료 + 긴 컨텍스트 |
| 대량 데이터 배치 처리 (API) | **Grok** | 2M 컨텍스트 + 최저 API 가격 |

### 콘텐츠 & 분석

| 상황 | 추천 | 이유 |
|------|------|------|
| 블로그/글 작성 | **ChatGPT** | 자연스러운 문체 + 이미지 생성 |
| 긴 문서/PDF 분석 | **Claude** or **Gemini** | 1M 컨텍스트, 정확한 요약 |
| 영상 분석/요약 | **Gemini** | 네이티브 영상 입력 |
| 실시간 트렌드 분석 | **Grok** | X 실시간 데이터 접근 |
| 심층 리서치 보고서 | **ChatGPT** (Pro) | Deep Research 기능 |

### 일상 & 생산성

| 상황 | 추천 | 이유 |
|------|------|------|
| 이메일/캘린더 정리 | **Gemini** | Google Workspace 직접 연동 |
| 일상 질문/대화 | **ChatGPT** | 가장 자연스러운 대화 + 음성 |
| 비용 최소화 | **Grok** (API) or **Gemini** (무료) | 최저가 또는 관대한 무료 |
| 수학/과학 문제 풀이 | **ChatGPT** (o3) | 추론 모델 특화 |
