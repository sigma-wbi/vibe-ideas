## 아이디어 요약

단순 돈 목적의 양산형 AI 쇼츠(REJECT) 대신, **김햄찌 모델처럼 고유 캐릭터 IP를 구축**하는 AI 쇼츠 채널 운영. 캐릭터의 페르소나·세계관·감정을 일관되게 유지하며 장기 팬덤을 쌓아 광고 수익 + 브랜디드 협업 + 2차 상품화까지 노린다.

**사용자 조건 (재확인):**
- 주당 5-10시간 (주말 집중)
- 월 30만원+ 예산 감당 가능
- **1년 이상 버틸 각오** (캐릭터 빌드업 기간 확보)
- 차별화 자산: **개그 감각 / 밈 감지 능력** + 자동화/개발 스킬 (후자는 차별화 아님)
- 캐릭터 유형: **아직 미정 → 조언 요청**

## 김햄찌가 성공한 진짜 이유 (Case Study)

> 사용자 질문의 핵심: "김햄찌는 왜 성공했고 카피캣은 왜 실패했나?"

### 1. 감정/결핍을 브랜드로 삼았다

- "정서불안"이라는 **감정 상태 자체**가 채널명/브랜드
- AI로 만든 귀여운 햄스터가 주인공이 아니라, **Z세대 직장인의 무기력·불안감**이 진짜 주인공
- 캐릭터는 그 감정의 아바타에 불과함

### 2. 구체적 페르소나 = "디자인 회사 다니는 암컷 햄스터 김햄찌"

- 단순 동물이 아니라 **직업·성별·이름·직장문화**를 가진 인물
- "디자인 회사"라는 구체적 설정이 무한한 에피소드 원천 (야근, 팀장 갈등, 포폴 스트레스 등)
- 시청자가 자기를 투영할 앵커 포인트 제공

### 3. AI 기술이 아니라 스토리텔링에 집중

- 업계 분석 공통 결론: **"AI로 만들었다는 게 놀라운 게 아니라, 공감되는 이야기가 놀라웠다"**
- 댓글에 "양산형 짤 같지 않다"는 반응 지속적으로 달림
- AI는 제작 수단일 뿐 마케팅 포인트가 아님

### 4. 30초 쇼츠 포맷 완벽 최적화

- 3초 훅 → 20초 빌드업 → 7초 반전/펀치라인
- 공감 + 유머 + 반전 + 귀여움이 한 영상에 압축
- 시청 완료율 높음 → 알고리즘 추천 배수

### 5. 캐릭터 일관성 (AI 난제 극복)

- AI 캐릭터 채널의 최대 약점: 매 영상마다 얼굴·체형·털색 미묘하게 달라짐
- 김햄찌는 이 지점에서 **"굉장히 높은 수준"**의 일관성 확보 (LoRA 학습/레퍼런스 시트 추정)

### 6. 인간의 목소리가 핵심 앵커

- 영상은 AI지만 **내레이션은 사람 목소리**
- 목소리 톤이 감정 공감의 주축 — 카피캣들은 여기서 무너짐

### 카피캣이 실패한 이유 (대응)

| 김햄찌 | 실패 카피캣 |
|--------|-------------|
| 감정 브랜드 ("정서불안") | "귀여운 햄스터 영상" (브랜드 없음) |
| 디자인회사 디테일 | 막연한 일상 |
| 스토리텔링 우선 | AI 기술 과시 |
| 인간 목소리 | AI TTS (몰입 깨짐) |
| 캐릭터 일관성 높음 | 영상마다 다른 얼굴 |
| 30초 완성도 | 늘어지거나 짧기만 함 |

**결론:** 양산형 카피캣은 껍데기(동물 캐릭터)만 베꼈고, 본질(감정·페르소나·스토리)을 놓쳤다.

## 5축 분석

### 1. 구현 가능성 — 7/10

**긍정:**
- 1년 각오 + 주당 5-10시간 + 월 30만원 = 캐릭터 빌드업 기간 충분
- 개발자 자동화 스킬 = AI 파이프라인 구축·버전 관리·스케줄링 효율 극대화
- Runway Gen-4.5의 캐릭터 일관성 기능으로 김햄찌 수준 구현 가능
- 음성: ElevenLabs 커스텀 보이스 또는 본인 녹음

**부정:**
- **캐릭터 컨셉 미정 = 가장 큰 리스크**. 이게 정해져야 나머지 진행
- 스토리 기획 역량이 제작 속도보다 병목 (주당 10-20개 에피소드 아이디어 필요)
- 캐릭터 일관성 유지는 여전히 AI 최대 난제 (LoRA 학습 등 추가 학습 필요)

### 2. 수익화 가능성 — 6/10

**데이터 (2026-04 기준):**
- 한국 쇼츠 RPM: 15-60원/1K views (광고 참여 증가로 과거 대비 현실화)
- 월 500만뷰 = 7-30만원 (예산 회수 가능선)
- 월 2000만뷰 = 30-120만원 (유의미한 부수입)
- **진짜 돈은 광고수익 아니라 브랜디드 협업**: 김햄찌 = 대기업 광고 다수 수주

**캐릭터 IP 단계별 수익 구조:**

| 단계 | 구독자 | 주 수익원 | 예상 월수익 |
|------|--------|-----------|-------------|
| 빌드업 | 0-1만 | 0 (YPP 미달성) | 0 |
| YPP 진입 | 1-10만 | 쇼츠 광고 | 5-30만원 |
| 팬덤 형성 | 10-50만 | 쇼츠 광고 + 소규모 브랜디드 | 30-200만원 |
| IP 확립 | 50만+ | 브랜디드 + 굿즈 + 롱폼 | 200만원~ 수천만원 |

**긍정:** 1년 각오면 YPP 진입-팬덤 형성 단계 도달 가능 (성공 시)
**부정:** 팬덤 형성 단계 미달성률이 80% 이상. 기대값 낮음

### 3. 시장 수요 — 6/10

**긍정:**
- 2026 유튜브 알고리즘이 **"recurring format" + "worldbuilding"** 채널을 선호
- 캐릭터 IP는 시청자가 장기 구독할 이유 제공 → 알고리즘 친화적
- 개그/밈 기반 콘텐츠는 Z세대 수요 지속

**부정:**
- 김햄찌 이후 "동물+직장인" 조합은 포화
- **카테고리 자체는 포화, 고품질 차별화 슬롯은 여전히 비어있음**
- 상위 10%만 수요 흡수

### 4. 차별화 — 5/10 ⚠️ (조건부 요소)

**긍정:**
- **사용자의 "개그 감각/밈 감지"는 김햄찌 성공 요인과 정확히 일치**하는 자산
- 1년 각오 = 장기 캐릭터 빌드업 가능
- 개발자 역량 + 개그 감각 조합은 드문 편

**부정:**
- **캐릭터 컨셉 미정** = 차별화의 핵심이 공백
- 자동화 스킬은 퀄리티가 아닌 속도에만 기여 → 시장에서 차별화 안 됨
- 기존 히트 컨셉(동물 직장인) 답습 시 카피캣 운명

**조건부 상향:** 컨셉 확정 + 본인 경험 기반 페르소나 확보 시 5 → 7로 상승 기대

### 5. 지속 가능성 — 5/10

**긍정:**
- **1년 이상 각오 = CONDITIONAL 판정의 결정적 근거**
- 캐릭터 IP는 축적될수록 가치 상승 (구독자·세계관 자산화)
- 팬덤 형성 시 플랫폼 리스크 일부 완화 (인스타/X로 팔로워 이전 가능, 김햄찌도 운영 중)

**부정:**
- 매주 10-15개 에피소드 = 스토리 고갈 리스크
- 캐릭터 번아웃 실사례 다수 (크리에이터가 캐릭터를 싫어하게 되는 현상)

## 추천 캐릭터 컨셉 (사용자 자산 기반 제안)

> 사용자 자산: **4년차 개발자 + 개그 감각/밈 감지 + 자동화 스킬**
> 김햄찌 공식: **감정 브랜드 + 구체 페르소나 + 본인 경험 기반 디테일**

### 컨셉 A: "번아웃 개발자 로봇" (최추천)

- **감정 브랜드:** "번아웃" / "무기력" / "AI에게 일 뺏기는 두려움"
- **페르소나:** 4년차 백엔드 개발자 로봇. 스타트업 다니다 구조조정 당하고 프리랜서 전향 고민 중
- **차별화:**
  - 김햄찌(디자인회사) 공식을 **IT 업계**로 옮김 → 포화 경쟁 회피
  - 본인의 실제 경험(구조조정, 업계 감정) = 무한 에피소드 원천
  - 2026 AI/개발자 시대정신과 정확히 맞물림
- **타겟:** 20-30대 개발자/IT업계 Z세대 (김햄찌 미커버 영역)
- **밈 코드:** GitHub 커밋, Jira 티켓, PR 리뷰, 번아웃, 재택, 퇴사 고민
- **시리즈 예시:**
  - "PR 리뷰 기다리는 중 정서불안 오는 로봇"
  - "3년만에 퇴사 고민하는 로봇의 1분 일기"
  - "AI가 내 일 뺏을까 불안한 로봇"

### 컨셉 B: "밈 중독 AI 소녀"

- **감정 브랜드:** "FOMO" / "도파민 중독" / "인터넷 의존"
- **페르소나:** 한국 인터넷 밈을 실시간 소비하는 AI 캐릭터. 인간 세상을 밈으로만 이해함
- **차별화:** 사용자의 "밈 감지 능력"을 정면 활용
- **리스크:** 밈 트렌드 빠르게 변함 → 에피소드 수명 짧음

### 컨셉 C: "퇴물 전자제품 의인화"

- **감정 브랜드:** "쓸모없어진 슬픔" / "아날로그 향수"
- **페르소나:** 낡은 MP3/폴더폰/초기 아이팟 등이 의인화되어 현대인을 풍자
- **차별화:** 전 세대 타겟 (부모 세대까지) / 감성 장르
- **리스크:** 개그보다 감성 비중 → 사용자 자산과 미스매치

**강력 추천:** 컨셉 A (번아웃 개발자 로봇). 본인 경험 + 차별화 자산 + 시대정신 3박자.

## 카피캣이 아닌 채널로 살아남는 5가지 체크리스트

> 김햄찌가 지킨 원칙을 본인 프로젝트에 적용하는 체크포인트

### ✅ 1. 감정/상태를 브랜드로 먼저 정의하라

- 캐릭터 이름 짓기 **전에** "내 채널의 감정 키워드"를 정해라
- 예: 정서불안 / 번아웃 / 도파민중독 / FOMO / 짜침
- 이게 먼저 안 나오면 시작하지 마라

### ✅ 2. 구체적 페르소나를 만들어라

- 나이·직업·회사·가족관계·취미까지 정한다
- "디자인 회사 햄스터"처럼 한 줄로 요약 가능해야 함
- 본인 실제 경험 베이스여야 에피소드가 안 고갈됨

### ✅ 3. AI 얘기를 홍보 포인트로 쓰지 마라

- 영상 설명·제목에 "AI로 만든" 절대 넣지 마라
- 시청자는 이야기에 빠져야지, 기술에 감탄하면 실패
- 시청자가 "이거 AI네"라고 알아채는 순간 몰입 끝

### ✅ 4. 인간 목소리를 써라 (최소 초기 1년)

- 본인 목소리 녹음이 가장 좋음
- 최소 ElevenLabs 커스텀 보이스 (TTS 기본 목소리 금지)
- 카피캣 실패 1순위 원인이 TTS 기본 목소리

### ✅ 5. 캐릭터 일관성 파이프라인에 투자하라

- 첫 3개월은 캐릭터 시트·레퍼런스 이미지 대량 제작
- Runway Gen-4.5의 character reference 기능 + LoRA 학습 고려
- 매 영상마다 얼굴 달라지면 팬덤 안 생김

## 추천 MVP 범위 (3개월 실험)

### 월 1: 캐릭터 확립

- 감정 브랜드 + 페르소나 확정 (1주)
- 캐릭터 레퍼런스 시트 10장 (2주)
- 파일럿 영상 3개 제작 (1주)

### 월 2: 업로드 리듬 구축

- 주 3개 업로드 시작 (총 12개)
- 인스타 동시 운영
- 댓글 분석 → 히트 포맷 식별

### 월 3: 히트 포맷 반복 + 확장

- 반응 좋은 포맷 집중 양산 (주 4-5개)
- 누적 50개 영상 / 1-3만 구독자 목표
- 이 단계에서 **명확한 Go/No-Go 판단**

### 성공 기준 (3개월차)

- ✅ GO: 구독자 1만+ / 평균 조회수 10만+ / 댓글 공감 우세
- 🟡 CONTINUE: 구독자 3천-1만 / 평균 3-10만 / 포맷 발견 중 → 3개월 연장
- ❌ PIVOT: 구독자 3천 미만 / 평균 3만 미만 → 컨셉 전면 재검토

- 쇼츠 광고 수익 월 5-30만원

### Phase 3 (12개월+): 본격 IP 수익화

- 브랜디드 협업 전담 (건당 수백-수천만원 가능)

- 굿즈 / 웹툰화 / 이모티콘 / 2차 창작 라이선싱

## 리스크 & 주의사항

### 치명적 리스크

1. **캐릭터 컨셉 확정 실패** — 이게 안 나오면 아무것도 시작되지 않음 (최우선 과제)
2. **초기 3개월 버티기** — 수익 0 + 반응 미미 구간에서 포기 시 모든 투자 증발
3. **카피캣 함정** — 김햄찌/진도설이 레시피 답습 시 자동으로 아류 판정

### 관리 가능 리스크

- 캐릭터 일관성 AI 이슈 → Runway + LoRA 학습으로 대응 가능
- 플랫폼 정책 변화 → 인스타/X 동시 운영으로 팬덤 이전 경로 확보
- 아이디어 고갈 → 본인 실제 경험 기반 페르소나면 1년치 에피소드는 가능

### 반드시 피해야 할 것

- ❌ "AI로 만들었다" 홍보 포인트 사용
- ❌ 동물 + 직장인 = 김햄찌 카피 (IT 업계로 각도 틀어야 함)
- ❌ TTS 기본 목소리
- ❌ 캐릭터 설정 없이 영상부터 찍기
- ❌ 3개월 만에 성과 판단 (최소 6개월)

### 핵심 발견 (원시)

- **김햄찌 성장:** 채널 개설 1개월만에 구독자 20-30만, 영상당 평균 100만뷰, 일일 구독자 2만 증가일
- **성공 5요소:** AI기술 + 공감 스토리텔링 + 매력적 캐릭터 + 센스있는 편집 + 인간 목소리
- **감정 브랜드화:** "정서불안" 자체가 브랜드 — 결핍/감정을 상품화
- **카피캣 전멸:** "동물이 인간사회에서 활동" 유사 컨셉 다수 생성 → "소리없이 사라짐"
- **2026-07-15 유튜브 정책:** "repetitive content" → "inauthentic content" 리브랜딩. 양산형 템플릿 수익화 불가
- **2026 쇼츠 RPM (한국):** 15-60원/1K views (과거 1-2원 대비 현실화). 광고주 참여 증가
- **알고리즘 선호:** recurring formats + worldbuilding + niche ownership

### 5축 점수 근거

- 구현가능성 7: 1년 각오 + 예산 + 개발 스킬. 캐릭터 일관성만 숙제
- 수익화 6: 1년 빌드업 시 팬덤 단계 도달 가능. 브랜디드가 본 수익
- 시장수요 6: 포화 but 고품질 차별화 슬롯 존재
- 차별화 5 (조건부): 개그/밈 감각 자산 존재. 컨셉 미정이 결정적 감점
- 지속가능성 5: 1년 각오 긍정 / 번아웃·고갈 리스크 상존
