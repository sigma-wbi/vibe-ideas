# docs 산출물 인덱스

> 워크플로우 스킬(/spec, /qa, /study 등)이 생성하는 문서가 카테고리별로 저장됩니다.
> index-sync Hook이 디렉토리별 _index.md를 자동 갱신합니다.

| 카테고리 | 용도 |
|----------|------|
| spec/ | 설계 문서 (/spec) |
| qa/ | QA·리뷰 결과 (/qa) |
| study/ | 학습 자료 (/study) |
| todo/ | 작업 목록 |
| knowledge/ | 지식 정리 (/reflect, /onboarding) |
| idea/ | 아이디어 검증 (/idea) |
